Inference Pipeline Notes

This page simulates a documentation site. It contains headings, table cells, inline formulas such as O(n log n), and code snippets that should remain untouched by translation.

The objective is to verify that structured technical content still reads well after a layout-preserving translation pass.

Example Pseudocode

function rankCandidates(scores) {
  return scores
    .map((value, index) => ({ value, index }))
    .sort((a, b) => b.value - a.value)
    .slice(0, 5);
}

The text around the code can be translated, but the identifiers scores, value, and index should not be rewritten into another language.
